Projects


Chocoholics Anonymous Software

In this project, my group created medical billing software for an imaginary health care provider called 'Chocoholics-Anonymous', written entirely in C++. For our primary task, we provided database functionality with patient info, provider info, services provided, and individual weekly invoices. We also provided APIs for a client-side application as well as for an external billing company to use to generate and send invoices to patients.

For testing, we created a driver program to simulate different use cases as well as the API handshake required to login from a client-side application. The driver program also simulates the main terminal that would be used at deployment for Chocoholics-Anonymous management to add and alter patient and provider records.

This software was created as a purpose-built application for a client complete with Specification and Requirements Documentation. We used a modified-waterfall approach to creating this software collaboratively and in a way that met all of our requirements. The whole project is written exclusively in C++, with an external JSON library for persistence.

Discrete Finite Automata Simulator

The DFA Simulator is a Python program designed to simulate the behavior of Discrete Finite Automata (DFAs). By providing a set of rules and a DFA graph, this simulator accurately predicts the output based on the given input sequence.

The DFA Simulator serves as a valuable tool for understanding, testing, and experimenting with DFAs. It provides a user-friendly interface, comprehensive error handling, and accurate simulation, making it an ideal choice for studentsx exploring the behavior of DFAs.
